# 💡 Ideas for Scrimbassadors
### 30 ideas across 4 categories | 11h 30min–12h 15min total

[Scrimbassadors](https://scrimba.com/scrimbassadors) are Scrimba students who help spread the word about learning to code on Scrimba by creating content, building in public, and growing the community. This list is the Scrimbassador idea bank. 🚀

## 🎁 Earn a Free Month of Scrimba Pro

Complete any 10 tasks, and you can claim a free month of Scrimba Pro! Here's how:

1. Add links to your completed tasks in your forked [GitHub](https://github.com) list as evidence
2. Message **@leannefromscrimba** on [Discord](https://scrimba.com/discord) to claim

> You'll need to be in the Scrimba Discord to claim. Join if you haven't already!

> Minimum time investment: approx 1h 40min. See the task list below to pick your 10!

---

## 🍴 How to Use This List

1. **Fork this repo first** (top right of this page). You'll need your own copy before the checkboxes will work
2. Open `README.md` and click the ✏️ pencil icon to edit
3. Check off tasks as you go by clicking the checkboxes or changing `[ ]` to `[x]`
4. Commit your changes and watch your progress grow!

- [x] 🌳 Add your Scrimba affiliate link to your [Linktree](https://linktr.ee) or bio links with a clear CTA *(5 min)*
- [ ] 📸 Update your [Instagram](https://www.instagram.com/codewithscrimba/) bio. Add "Learning to code with @codewithscrimba" or similar 🌟 *(5 min)*
- [ ] 💼 Update your **LinkedIn** headline. Add something like "Teaching others to code | Scrimba Student" *(5 min)*
- [ ] 🎠 Create an [Instagram](https://www.instagram.com/codewithscrimba/) Highlight reel. Save your Scrimba-related stories to a "Coding" or "Scrimba" highlight *(5 min)*
- [ ] 📌 Pin a Scrimba post to the top of your [X/Twitter](https://twitter.com/scrimba) profile *(10 min)*
- [x] 👩‍💻 Set up a [Dev.to](https://dev.to/grindstack) account. Add a bio mentioning you're learning with Scrimba *(15 min)*
- [ ] ✍️ Set up a [Hashnode](https://hashnode.com) blog. Customize your theme and write an intro post about your coding journey *(30 min)*
- [x] 🐙 Update your [GitHub](https://github.com/grindstack) profile README. Add a section about your learning journey and the Scrimba courses you've completed *(20 min)*
- [x] 📝 Create a [Medium](https://medium.com/@fatymah) account. Set up your profile with a Scrimba mention and follow relevant coding publications *(15 min)*
- [ ] 🎬 Set up a [YouTube](https://www.youtube.com/@scrimba) channel. Create or optimize it for coding content and add Scrimba to your channel description *(20 min)*

## 🎨 Content Creation. 14 tasks | 6h 40min–7h 10min

### 📺 YouTube. 4 tasks | 1h 25min–1h 40min
- [ ] 🎥 **Short:** "How I'm learning [language/framework] with Scrimba." 60 seconds, include your discount code in the description *(20 min)*
- [ ] 🖥️ **Tutorial:** Screen record yourself working through a Scrimba concept and mention Scrimba in the title *(30–45 min)*
- [ ] 📈 **Progress update:** Show off projects, certificates, or skills you've gained on Scrimba *(20 min)*
- [ ] 💡 **Quick tip:** Share a coding tip from today's Scrimba lesson *(15 min)*

### 📰 Dev.to / Hashnode / Medium. 6 tasks | 4h 15min–4h 30min
- [ ] 📖 **"What I learned today"** Write about today's Scrimba lesson with code snippets *(30 min)*
- [ ] 🧪 **Tutorial post.** Turn a Scrimba lesson into a written tutorial with your own examples *(45–60 min)*
- [ ] 🤔 **"Why I chose Scrimba"** Write about your decision to learn with Scrimba vs. alternatives *(30 min)*
- [ ] 🗺️ **"My coding journey so far"** A progress update mentioning the Scrimba courses you've completed *(30 min)*
- [ ] 📋 **Cheat sheet.** Create a helpful [language] reference, mentioning you learned these concepts on Scrimba *(1h)*
- [ ] 🏗️ **"Building my first [project type]"** Document a project you built using skills from Scrimba *(1h)*

### 📱 Social Media. 4 tasks | 1h
- [ ] 🎞️ [Instagram](https://www.instagram.com/codewithscrimba/) Reel: Short code-along video with a text overlay explaining the concept *(20 min)*
- [ ] 📸 [Instagram](https://www.instagram.com/codewithscrimba/) Story: Screenshot of your Scrimba lesson + what you learned. Add your discount code sticker *(10 min)*
- [ ] 🎵 [TikTok](https://www.tiktok.com/@scrimba): "Follow my coding journey." Show your progress and mention Scrimba *(15 min)*
- [ ] 💼 **LinkedIn:** Share a professional progress post about career or skill development wins from learning on Scrimba *(15 min)*

## 🗂️ Portfolio & Projects. 4 tasks | 1h 50min–2h 5min

- [ ] 🐙 Push a [GitHub](https://github.com) project with a Scrimba mention in the README. Credit the course that taught you the skills *(30 min)*
- [ ] 👩‍💻 Write a [Dev.to](https://dev.to/scrimba) post showcasing a project you built with Scrimba skills *(45–60 min)*
- [ ] 🌟 Share your portfolio with **@leannefromscrimba** on [Discord](https://scrimba.com/discord) to potentially be featured in the Scrimba newsletter (don't forget to mention Scrimba and add your affiliate link!) *(15 min)*
- [ ] 💼 Post a **LinkedIn** project spotlight with a link to your [GitHub](https://github.com) or live demo, and mention your Scrimba learning *(20 min)*

## 🤝 Community Engagement. 2 tasks | 50min

- [ ] 💬 Leave helpful comments on 3–5 beginner coding [YouTube](https://www.youtube.com/@scrimba) videos, mentioning your Scrimba experience where relevant *(20 min)*
- [ ] 🙋 Find 3–5 [Dev.to](https://dev.to/scrimba) posts from beginners asking for advice. Share helpful tips and your learning experience *(30 min)*
